# 第2章 快速入门

## 2.1 安装Python

为了让计算机能够执行 Python 代码，我们需要 Python 解释器，从官网上下载的 Python 中内置解释器。

- Python 官网地址：https://www.python.org/

**安装步骤：**

1. 进入官网，点击 Downloads，选择对应的操作系统。
2. 选择版本，点击链接下载（课程版本为 **3.12.8**）。

这里主要有两类安装包：

### "install" 安装包

最常见的用于在桌面系统、服务器等常规环境中完整安装 Python 的方式。通常包含完整的 Python 标准库、解释器以及一些辅助工具（如 pip 用于安装第三方库）等。安装过程中会自动设置系统环境变量 PATH。

### "embeddable" 可嵌入安装包

主要设计用于将 Python 嵌入到其他应用程序中。它只包含最核心的 Python 解释器以及一些必要的基础组件，不包含完整的标准库，文件大小通常只有几兆。

> 综上所述，选择使用哪种 Python 安装包取决于具体的需求。我们属于搭建通用开发环境，所以选择 **Install 安装包**。

**继续安装步骤：**

3. 双击下载好的文件，开始安装。
4. 保持默认，点击 Next。
5. 修改安装路径，其他保持默认，点击 Install 开始安装。
6. 点击 **Disable python length limit**（禁用系统的路径长度自动限制，以避免因路径过长而导致的错误），点击 Close，完成安装。
7. 安装完成后检查是否安装成功：同时按下 `Win` 键和 `R`，输入 `cmd`，点击确定，进入命令提示符。
8. 输入 `python --version`，打印出 Python 版本，安装成功。

## 2.2 安装PyCharm

### 2.2.1 什么是IDE

集成开发环境（简称：IDE；英文名：Integrated Development Environment）是用于提供程序开发环境的应用程序，一般包括代码编辑器、编译器、调试器和图形用户界面等工具。集成了代码编写功能、分析功能、编译功能、调试功能等多种功能。

虽然我们也可以使用记事本编写代码，并通过命令行调用 Python 解释器来执行 Python 程序。但这样比较繁琐，会降低开发效率。而使用 IDE 后，很多工作可以交给 IDE 帮我们去做，让我们可以专注于代码的编写。

Python 的 IDE 我们选择使用 **PyCharm**。

### 2.2.2 PyCharm 下载

- PyCharm 官方地址：https://www.jetbrains.com/pycharm/download
- 进入官网，下载 PyCharm 安装包，专业版可试用 30 天，社区版完全免费。

### 2.2.3 PyCharm 安装

1. 双击安装包进入安装。
2. 点击下一步。
3. 修改安装目录，点击下一步。
4. 酌情勾选安装选项，之后点击下一步。
5. 点击安装。
6. 安装完成。

### 2.2.4 PyCharm 设置

#### 1）创建新项目

1. 点击 **New Project** 创建新项目。
2. 设置项目名称、项目路径、解释器类型、Python 版本。

> 注意：不同的 PyCharm 版本，看到的界面会略有不同。

**解释器类型说明：**

- **Project venv**：当前项目的虚拟环境，Python 版本可以在右侧小箭头下拉选择，在当前项目内安装的包只会在项目内有效，在项目目录下会有一个 `.venv` 目录。
- **Base Conda**：Anaconda 环境，如果选择这个需要先在电脑内安装 Anaconda。
- **Custom environment**：自定义环境，可以：
  - **generate new**：新建虚拟环境（选择本机已安装的 Python 或下载新的 Python 作为 Base Python），生成后放在其项目文件夹下的 venv 中。所有需要使用的第三方库都需要重新安装。
  - **Select existing**：选择已配置好的虚拟环境作为自己的编译器，所选择的虚拟环境中的所有已安装的第三方包都可以使用。

3. 一个 Python 项目创建成功。

#### 2）主题设置

打开设置面板，进入 `Appearance & Behavior` → `Appearance`，修改主题。

#### 3）字体设置

进入 `Editor` → `Font` 设置字体。

#### 4）中文设置（建议使用默认的 English）

1. 首先确认已经安装了中文语言包。
2. 切换简体中文。

#### 5）快捷键设置

在 `Keymap` 下可对快捷键进行设置。常用快捷键：

## 2.3 第一个Python程序

### 2.3.1 Python 程序运行方式

有多种方式执行 Python 程序，以下是常见的三种方式：

- 交互式命令行模式
- 脚本模式
- 集成开发环境（IDE）模式

### 2.3.2 交互式命令行模式

1. 同时按下 `Win` 键和 `R`，并输入 `cmd`，进入命令提示符。
2. 在命令提示符中输入 `python`，进入 Python 交互模式。
3. 输入 `print("hello")`，按下回车，控制台会打印 `hello`。

### 2.3.3 脚本模式

1. 在 `E:\Hello` 路径下新建一个文件，将其重命名为 `hello.py`。
2. 双击此文件，选择使用记事本打开，在其中写入 `print("hello")`，并保存。
3. 在资源管理器上方输入 `cmd` 并回车，就会打开命令提示符并进入当前路径。或先打开命令提示符，再输入 `E: && cd Hello` 进入 `E:\Hello` 路径。
4. 在命令提示符中输入 `python hello.py` 执行程序。

### 2.3.4 IDE 模式

使用 IDE 运行 Python 程序，也是日后我们最常用的方式。

1. 在 python-2025 项目下创建一个新的目录 `chapter02`。
2. 在 chapter02 目录下创建一个 Python 程序文件，输入：

```python
print("hello world")
```

3. PyCharm 工作区空白处右键，单击 `Run 'xxx'`。
4. 可以在下方控制台看到程序执行结果，打印出了 `hello world`。
